# Job skillsets

### Types of Software and Technologies

- Software engineers develop various software including desktop, web, mobile apps, games, operating systems, and network controllers.
- They use technologies such as programming languages, development environments, frameworks, libraries, databases, and servers.

### Categories of Software Engineers

- Back-end engineers (system developers) build computer systems and networks that support front-end applications.
- Front-end engineers (application developers) create user-facing software like mobile apps and websites.

### Day-to-Day Tasks and Responsibilities

- Tasks include designing software based on user specifications, writing and testing code, optimizing performance, maintaining existing software, and documenting code.
- Responsibilities grow with experience, from coding and testing in junior roles to planning and designing multiple software areas in senior roles.

## A Day in the Life of a Software Engineer

### Daily Work Routine

- The engineer starts the day by checking messages, calendar, and code merge requests, receiving feedback and suggestions from team members and mentors.
- Daily standup meetings involve team members sharing progress and plans, with mentors providing advice on code optimization.

### Coding and Collaboration

- Focus time is dedicated to improving code performance based on feedback and running tests to ensure functionality.
- Collaboration with other departments, such as marketing, introduces new feature requests and opportunities to learn new technologies like React.

### Learning and Problem Solving

- Continuous learning is emphasized through self-study of documentation, videos, and blogs to gain new skills.
- Bug reports are addressed promptly by identifying issues, fixing code, and submitting updates for review, highlighting the importance of testing and quality assurance.

### Hard Skills for Software Engineers

- Hard skills are technical, practical skills learned through education or experience, such as programming languages, version control, cloud computing, testing, debugging, and database architecture.
- Key hard skills include software analysis and design, programming in languages like Java, Python, C#, and Ruby, understanding frameworks and object-oriented principles, testing, debugging, deployment, monitoring, and troubleshooting.

### Soft Skills for Software Engineers

- Soft skills are personal and interpersonal traits that are less tangible and harder to quantify but transferable across roles and industries.
- Important soft skills include teamwork, communication with diverse stakeholders, time management, problem-solving, adaptability to changing project needs, and openness to feedback for continuous improvement.
